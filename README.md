# 🌆 Night City Cyberpunk Edgerunners

A **Cyberpunk 2077 inspired VS Code theme**, focused on neon visuals, immersive dark UI, and a futuristic coding atmosphere.

Inspired by:

- Cyberpunk 2077
- Night City
- Arasaka aesthetics
- Cyberpunk: Edgerunners

---

# ✨ Features

### 🎨 Custom UI Colors
- Neon cyan highlights
- Cyberpunk yellow accents
- Dark immersive background
- Customized activity bar, sidebar, tabs, widgets, menus and status bar

### 💻 Syntax Highlighting
Custom token colors for:

- JavaScript
- TypeScript
- HTML
- CSS
- JSON
- Markdown
- Python

### 🌃 Optional Night City Wallpaper
An optional wallpaper setup is available for a full immersive cyberpunk experience.

## Manual Installation (.vsix)

1. Download the latest `.vsix`
2. Open VS Code
3. Press:

```txt
Ctrl + Shift + P
```

4. Search:

```txt
Extensions: Install from VSIX
```

5. Select the extension file

6. Open:

```txt
Preferences: Color Theme
```

7. Select:

```txt
Night City Dark
```

# 🌆 Optional Wallpaper Setup

This theme supports an optional wallpaper experience.

## Step 1 — Install Extension

Install:

**Custom CSS and JS Loader**

https://marketplace.visualstudio.com/items?itemName=be5invis.vscode-custom-css

---

## Step 2 — Open settings.json

Press:

```txt
Ctrl + Shift + P
```

Search:

```txt
Preferences: Open Settings (JSON)
```

Add:

```json
"vscode_custom_css.imports": [
  "file:///YOUR_PROJECT_PATH/css/cidade.css"
]
```

Example (Windows):

```json
"vscode_custom_css.imports": [
  "file:///C:/Users/Luiz/Desktop/night-city-theme/css/cidade.css"
]
```

---

## Step 3 — Enable CSS

Press:

```txt
Ctrl + Shift + P
```

Run:

```txt
Enable Custom CSS and JS
```

Then:

```txt
Developer: Reload Window
```
